# 🐍 Modern Snake Game

**Modern Snake Game** — это интерактивная игра “Змейка”, сделанная полностью на **HTML, CSS и JavaScript**.  
Проект разработан как часть моего **портфолио**, чтобы продемонстрировать навыки веб-разработки и работу с **Canvas, анимациями и логикой игры**.

## 🎮 Как играть
- Используйте **стрелки** (⬅️⬆️➡️⬇️) для управления змейкой.  
- Ешьте еду (круглые жёлтые элементы), чтобы змейка росла.  
- Избегайте столкновения со стенами и с самим собой.  
- Очки отображаются в верхней части экрана.

## ⚡ Особенности проекта
- Современный дизайн с **градиентами и плавными тенями**  
- **Canvas** для отрисовки змейки и еды  
- Реалистичная логика змейки (рост, столкновения)  
- Простое и красивое управление  
- Адаптивная доска под любой размер экрана

## 🛠 Технологии
- HTML5  
- CSS3

# 🐍 Modern Snake Game

**Modern Snake Game** is an interactive Snake game built using **HTML, CSS, and JavaScript**.  
This project was created as part of my **portfolio** to demonstrate my frontend development skills, including working with **Canvas, animations, and game logic**.

---

## 🎮 How to Play
- Use **arrow keys** (⬅️⬆️➡️⬇️) to control the snake  
- Eat the food (yellow circle) to grow longer  
- Avoid hitting the walls or yourself  
- Your score is displayed at the top  

---

## ⚡ Features
- Modern UI with **gradients and smooth shadows**  
- Built using **HTML5 Canvas**  
- Realistic snake movement and collision logic  
- Clean and responsive layout
