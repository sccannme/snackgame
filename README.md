# 🐍 Modern Snake Game

**Modern Snake Game** — это интерактивная игра “Змейка”, сделанная полностью на **HTML, CSS и JavaScript**.  
Проект разработан как часть моего **портфолио**, чтобы продемонстрировать навыки веб-разработки и работу с **Canvas, анимациями и логикой игры**.

## 🎮 Как играть
- Используйте **стрелки** (⬅️⬆️➡️⬇️) для управления змейкой.  
- Ешьте еду (круглые жёлтые элементы), чтобы змейка росла.  
- Избегайте столкновения со стенами и с самим собой.  
- Очки отображаются в верхней части экрана.

## ⚡ Особенности проекта
- Современный дизайн с **градиентами и плавными тенями**  
- **Canvas** для отрисовки змейки и еды  
- Реалистичная логика змейки (рост, столкновения)  
- Простое и красивое управление  
- Адаптивная доска под любой размер экрана

## 🛠 Технологии
